CSS 变量映射（实际实现）

元素 CSS 变量 暗色效果 亮色效果
对话框背景 --b3-theme-surface 深色面板 白色面板
主文字 --b3-theme-on-surface 浅灰白 深灰黑
次要文字 --b3-theme-on-surface-light 灰色 中灰
边框 --b3-border-color 深灰线 浅灰线
用户气泡 --b3-theme-primary-lightest 深蓝底 浅蓝底
AI 气泡 --b3-theme-surface-light 稍深灰 浅灰底
按钮 / 链接 --b3-theme-primary 主题蓝 主题蓝
来源标签 --b3-theme-success 绿色标记 绿色标记
实现策略： 全部使用 var(--b3-theme-*) 变量，零硬编码颜色。如需气泡等细微差异，用 :global(.b3-theme-dark) / :global(.b3-theme-light) 选择器覆盖局部变量，与 SummaryCardsGrid.vue 保持一致。
