# 脉络镜 —— 让隐没的知识，重现脉络

## v0.9.6
### 新增
- LLM Wiki 维护输出内容支持Markdown常用格式渲染；支持diff显示；
- “已读文档”详情支持一键添加已读标记；
### 改进
- 设置页选项支持分区折叠和展开，可选项默认折叠；
- UI细节优化；

## v0.9.5
### 新增
- LLM Wiki 预览版
  - 基于主题笔记和关联文档生成 LLM Wiki，包括Wiki、索引、日志页面；
  - LLM Wiki 统计卡片；
  - 基于 LLM Wiki 信息进行 AI 对话；
  - AI 核查 LLM Wiki，并提供维护更新建议； 
- 设置页调试区新增查看文档索引开关；
- 文档索引支持批量生成和删除；
### 改进
- 大文件字数统计方式优化；
- 配置内容不即时生效，需刷新后生效；
- 取消Embedding模型配置；
- 孤立文档AI建议支持推荐新标签；
- 核心文档排名纳入出链与子文档数量，统一连接度排序；
- 大量UI细节优化；

## v0.8.6
### 新增
- 首个英文预览版；
- “今日建议”保留最近 3 次 AI 分析结果；
- 新增设置：分析范围（排除路径）、日志开关；
### 改进
- 主题文档标题前缀和后缀不纳入关联分析；
- UI细节优化；

## v0.8.0

- 支持对“孤立文档”提供建立连接和打标签的建议（实验性）；
### 改进
- UI细节优化；
